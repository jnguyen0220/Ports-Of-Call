# Ports Of Call
A real time heartbeat web application that uses cron syntax for scheduling.

## Demo
[http://71.147.4.114:3000](http://71.147.4.114:3000)

## Prerequisites 
```
git
nodeJS
Docker
Docker-compose
```
## Installing
```
git clone https://github.com/jnguyen0220/Ports-Of-Call"
npm install
```

## Running 

### Local
```
npm start
```
### Docker
```
docker-compose up -d --build
```
